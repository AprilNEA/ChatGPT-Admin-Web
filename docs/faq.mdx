## 1. 如果本地部署该项目？
可以查询如果本地部署 Next.js 项目。
环境变量可以在 `apps/chat` 文件夹中放置 `.env` 文件。

## 2. 后台管理面板在哪？

后台管理的源码位于 `apps/dash` 中，需要单独进行部署。
目前正在重新开发中，请耐心等待。

## 3. Redis 是否可以不使用 Upstash ？
我们正在使用 Postgresql 或本地 Redis 来取代 Upstash，请耐心等待。

## 4. 如何使用 Docker 进行部署？
本项目暂时没有构建官方 Docker 镜像。

##5. 支付接口对接无效/邮件收不到验证码怎么办？
确认环境变量的键值均配置无误时，可携带详细的控制台报错(截图/文字)前往 Github 发起 issue。

# 接入手机号验证

| 键         |         值 |
|:------------:|:-----------:|
| SMS_SERVICE | 目前仅支持"uni" |


## UniSMS
![](/install/unisms-01.png)

UniSMS 是一个聚合短信服务平台。

<Warning>
    申请签名时请注意，如果需要向国内用户发送短信，所申请的签证名中必须包含中文。
</Warning>

环境变量如下：

| 键                |                                 值 |
|:------------------:|:-----------------------------------:|
| SMS_UNI_KEY_ID    |                       UniSMS 的 key |
| SMS_UNI_SIGNATURE | 在 UniSMS 中申请的签名，发送+86的短信请保证签名中带有中文 |

# 配置邮件服务器

目前我们仅接入支持通过`Restful API`来发件的第三方发件服务，需要自行注册后获取相关的密钥，并填入环境变量。

## 通用环境变量

| 环境变量的名称(NAME)             | 环境变量的内容(KEY)          | 备注                |
|---------------------------|-----------------------|-------------------|
| NEXT_PUBLIC_EMAIL_SERVICE | `elastice`或者`mailgun` | 留空或不填写改变量则不启用邮箱验证 |
